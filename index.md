## À propos

Vous avez besoin de reprendre une application existante, lancer rapidement un prototype fiable, ou faire évoluer un produit web devenu difficile à maintenir ?

J’interviens comme développeur full stack senior pour concevoir, refondre et faire avancer des applications web utiles au métier, avec un fort niveau d’exigence sur la qualité du code, la maintenabilité et la vitesse d’exécution.

Mon socle principal est Python / Django côté backend, avec React et Vue.js côté frontend. J’interviens aussi sur l’environnement de delivery avec Docker, Docker Compose et la mise en place de pipelines CI/CD via GitHub Actions ou GitLab CI.

J’aide mes clients à :
- Concevoir un prototype propre et exploitable rapidement
- Reprendre une base existante pour la rendre plus lisible et plus maintenable
- Mener une refonte progressive sans bloquer le métier
- Accélérer les développements grâce à de meilleures fondations techniques
- Industrialiser les déploiements et fiabiliser les livraisons
- Documenter, tester et structurer le produit pour tenir dans la durée

J’ai travaillé dans des contextes startup, SaaS, logiciels métiers, banque, transport, santé et enseignement supérieur. Mon objectif : livrer une solution utile, lisible et durable.

J’utilise également l’IA de manière professionnelle, notamment avec GitHub Copilot, pour accélérer le développement et améliorer la qualité des livrables.
